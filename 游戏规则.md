# 大富翁中国行 v2 - 在线对战版 游戏规则

## 游戏概述

大富翁中国行 v2 是在线对战版，支持 2-4 人通过房间码匹配进行实时在线对战。无需注册账号，输入昵称即可游戏。玩家通过掷骰子移动、购买地皮、收取租金来积累财富，最终成为最富有的玩家。

---

## 联机系统

### 创建/加入房间

| 步骤 | 说明 |
|------|------|
| 输入昵称 | 最多 8 个字符 |
| 创建房间 | 选择人数（2-4人）和初始资金 → 获得 4 位房间码 |
| 加入房间 | 输入房间码 → 进入等待室 |
| 准备 | 所有玩家（含房主）点击"准备" |
| 开始 | 全部准备后，房主点击"开始游戏" |

### 房间规则

- 房间码由 4 位字母+数字组成（如 A3K7）
- 房间 5 分钟无活动自动销毁
- 玩家断线 → 自动判定破产并退出
- 房主离开 → 自动转移房主给下一位玩家

### 通信机制

- WebSocket 实时同步（Socket.IO）
- 服务端权威状态：所有游戏逻辑在服务端执行，客户端只发送操作指令
- 操作指令：掷骰子、购买、跳过、变卖、交易、拍卖出价

## 回合流程

1. **掷骰子**：轮到你时点击"掷骰子"，投掷两颗骰子（各 1-6 点）
2. **移动**：按点数总和前进相应格数
3. **经过起点**：获得 ¥200 奖励（随回合递增，每 5 回合 +¥50，上限 ¥400）
4. **执行操作**：根据到达格子的类型执行对应操作
5. **等待**：非你的回合时显示"等待 XX 操作..."

### 回合前可选操作

在掷骰子之前，玩家可以：
- **变卖资产**：6 折出售自己的地皮换取现金
- **发起交易**：向其他玩家提出购买其地皮的请求

## 购买规则

到达无主地皮时：
- **余额充足**：弹出购买/跳过选择
- **余额不足**：自动跳过，显示"资金不足"
- **放弃购买**：触发拍卖，其他玩家可竞价

---

## 拍卖规则

当玩家放弃购买无主地皮时，自动触发拍卖：

1. **起拍价**：地皮原价的 50%
2. **加价**：每次 +¥20 或 +¥50
3. **轮流竞拍**：每位未放弃的玩家可选择加价或放弃
4. **结束条件**：所有人放弃后，最高出价者获得地皮
5. **流拍**：无人出价则地皮保持无主

---

## 交易规则

玩家可在自己回合掷骰前发起交易：

1. **选择对象**：选择有地皮的其他玩家
2. **选择地皮**：从对方地皮中选择想购买的
3. **出价**：选择报价金额（原价 x1.0 / x1.3 / x1.5）
4. **对方响应**：对方收到交易请求弹窗，选择接受或拒绝

---

## 变卖资产

玩家可在自己回合掷骰前变卖地皮换取现金：

- **折扣**：6 折出售（原价 x0.6）
- **用途**：缓解资金压力、避免破产

## 监狱规则

**入狱方式**：到达"入狱"格 或 机会卡"进监狱"

**出狱方式**：
- 掷出双数（两骰子点数相同）→ 出狱并按该点数移动
- 等待 3 回合 → 缴纳 ¥50 保释金自动出狱

---

## 破产规则

当玩家资金变为负数时：
1. 自动从最便宜的地皮开始 6 折出售
2. 直到资金 >= 0 或地皮卖光
3. 卖光仍为负 → 破产退出
4. 现金为 0 且无地皮 → 同样破产

**断线处理**：玩家断线 → 自动判定破产

---

## 胜利条件

满足以下任一条件即获胜：

1. **淘汰胜利**：其他所有玩家破产
2. **资产胜利**：30 回合结束，总资产最高者获胜

**总资产** = 现金 + 地皮原价总和
